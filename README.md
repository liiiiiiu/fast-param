# fast-param

⚡️ Vue3 + Vite + Typescript ⚡️ 开发的一款工具，用于前端快速生成不同业务场景下的对象属性。

![fast-param](https://s2.loli.net/2022/07/18/dWvLPaGgKp5JYs7.png)

[在线预览](https://fastparam.liiiiiiu.com/)

## 使用

```bash
git clone https://github.com/liiiiiiu/fast-param.git

cd fast-param

npm install

# 本地访问
npm run dev

# 打包发布
npm run build
```
